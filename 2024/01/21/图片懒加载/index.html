图片懒加载

图片懒加载是一种优化策略，核心思想在于进入视口区域的图片才进行加载，本文总结图片懒加载的三种方式。

Element.getBoundingClientRect()

Element.getBoundingClientRect() 方法返回元素的大小以及相对视口的位置的对象。

返回的对象有以下属性：

  • x 元素左侧到视口的水平距离
  • y 元素上方到视口的竖直距离
  • width 元素的 width + padding
  • height 元素的 height + padding
  • top 同 y
  • right 元素的右侧到视口的水平距离
  • bottom 元素的下方到视口的竖直距离
  • left 同 x

懒加载函数：

开发中监听滚动事件监听此函数，实现图片懒加载，注意使用节流。

IntersectionObserver

IntersectionObserver 提供了一种异步观察元素与其视口交叉状态的方法，使用该方法得知图片元素是否进入视口，从而实现图片的懒加载。

语法：

options 选项

  • options.root： 当做视口的元素，默认是浏览器视口
  • options.rootMargin： 计算交叉点前，触发的边距
  • options.threshold： 触发回调的百分比，一个数值或者多个数值的数组，如果 .5 则表示元素能见度超过 50%时触发回调

callback 触发可视的回调，会传入两个参数

  • entries： IntersectionObserverEntry对象组成的数组
  • observer： IntersectionObserver实例

使用 IntersectionObserver 实现图片懒加载

loading 属性

设置图片的 loading 属性

  • eager 立即加载图像
  • lazy 延迟加载图像，直到它与视口接近一个距离
