动画实现方式

整理前端网页实现动画的方式。

transition(过渡)

css transition 用来定义元素的属性变化方式，让其属性变化持续一段时间。
属性：

  • transition-property 指定过渡的属性
  • transition-duration 过渡的时长
  • transition-timing-function 属性变化的函数
  • transition-delay 属性变化的延迟时间

开发中与过渡相关的问题，如下图，内容不确定的情况下对高度做一个过渡效果。height 属性值为 auto 时，容器高度自动展开，但 auto 不能触发过渡效果，所以我们需要计算容器高度。

animation(动画)

通过 css animation 实现动画
属性：

  • animation-name 动画名称
  • animation-duration 动画持续时间
  • animation-timing-function 动画周期函数
  • animation-delay 动画延迟时间
  • animation-iteration-count 动画播放次数
  • animation-direction 动画播放的方向
  • animation-fill-mode 动画填充模式
  • animation-play-state 动画播放状态

requestAnimationFrame

使用 requestAnimationFrame 来制作动画。

  • requestAnimationFrame 比定时器更适合用来做动画效果，它会在浏览器下一次重绘之前执行传给它的回调函数，它的执行次数通常与屏幕刷新率匹配。
  • 在后台标签页或者隐藏的 iframe 里，requestAnimationFrame 会被暂停以提高性能。

Web Animation API

使用Element 的 animate() 方法，创建动画。
语法：element.animate(keyframes,options) ，接收两个参数，第一个参数 keyframes ：关键帧对象 ，第二个参数 options ： 动画持续时间或者包含多个动画选项配置的对象。
